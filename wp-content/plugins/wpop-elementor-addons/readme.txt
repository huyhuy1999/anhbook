== Description ==
This is a plugin for WPoperation WordPress Themes. The theme contains elements of elementor.


=== Features === 

Adds feature to WPoperation WordPress Themes by adding custom elements
Translation Ready
Lifetime Free support
Constant updates

== Installation ==
1. Unzip wpop-elementor-addons.zip
2. Upload all the files to the /wp-content/plugins/wpop-elementor-addons
3. Activate the plugin through the 'Plugins' menu in WordPress.


== Frequently Asked Questions ==
= What does this plugin do? =
This plugin adds some elements of elementor builder to configure the Opstore theme.

= Will it work after installing and activating? =
This plugin is addon for elementor, so elementor plugin must be installed and activated to use this.
