== Description ==
This is the demo importer plugin for WPOperation themes.


=== Features === 

Adds Demo Import Functiolanity to the WordPress Themes By WPOperation
Translation Ready
Lifetime Free support
Constant updates

== Installation ==
1. Unzip wpop-demo-importer.zip
2. Upload all the files to the /wp-content/plugins/operation-demo-importer
3. Activate the plugin through the 'Plugins' menu in WordPress.

== Frequently Asked Questions ==
= What does this plugin do? =
This plugin will add feature to import demos to the themes by WPOperation
